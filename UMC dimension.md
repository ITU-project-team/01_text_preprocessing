| Dimension Key          | 설명                                                                                                                |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `connection_quality`   | 4G/5G 기지국 밀도와 평균 다운로드 속도로 측정하는 통신 인프라 수준                                                  |
| `availability_for_use` | 모바일 데이터 사용량, 온라인 서비스 이용일수, 공공 WiFi 밀도 등 위치에 구애받지 않고 디지털 서비스에 접근 가능 정도 |
| `affordability`        | 가구 평균 소득과 통신비 체납률로 측정하는 디지털 서비스의 경제적 부담 수준                                          |
| `devices`              | 가정 내 핵심 기기(PC·노트북·태블릿) 보유 수와 개인 사용 기기 수로 측정하는 디지털 기기 접근성                       |
| `digital_skills`       | 스마트기기 기본 조작, 디지털 서비스 활용, 스마트기기 활동 숙련도, 사이버 활동 능력을 포괄하는 디지털 활용 역량      |
| `safety_and_security`  | 보안 행동(백신·업데이트·비밀번호 변경 등)과 보안 인식(사이버 범죄 대응 인지)으로 측정하는 디지털 안전 수준          |
